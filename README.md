# Image Search

A responsive image search web application built using **HTML**, **CSS**, **JavaScript**, and the **Unsplash API**. The application allows users to search and browse high-quality images based on keywords with a clean and responsive user interface.

## Features

- Search images using the Unsplash API
- Display high-quality images based on user queries
- Responsive design for desktop and mobile devices
- Fast and user-friendly interface
- Loads unique and relevant images for each search

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Unsplash API
